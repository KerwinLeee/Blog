## 原生js
#### 1.数组合并的常用两种方法.
```
// 1.concat，可以合并多个数组，不会影响原数组，但不能处理嵌套数组
var a = [1, 2, 3], b = [4, 5, 6];
a = a.concat(b);
document.write(a)

// 2.apply，可以处理嵌套数组
arr1 = [1,2,3,[ 'a','b']]; arr2 = [ 'a','b'];
function concat_apply() {
	// return  Array.prototype.push.apply(arr1,arr2);
	return Array.prototype.concat.apply([], arguments)
}
```
