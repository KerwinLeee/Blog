### python相关笔记


#### 1.python3安装
```
brew install python

注意后面使用命令python3,包管理pip3...
路径：/usr/local/lib/python3.6/site-packages
```

#### 2.redis

首先安装wget
```
brew install wget
```

然后下载redis包
```
$ wget http://download.redis.io/releases/redis-4.0.11.tar.gz
$ tar xzf redis-4.0.11.tar.gz
$ cd redis-4.0.11
$ make
```

启动服务端
```
$ src/redis-server
```

启动客户端
```
src/redis-cli
```
