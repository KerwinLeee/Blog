
## 基本命令
#### 1.显示所有数据库列表
```
show databases;
```
#### 2.建库
```
create database Mytest;
```

#### 3.删库
```
drop database 库名;
```

#### 4.选择库
```
use 库名;
```

#### 5.打开某个数据库（比如数据库：Mytest)
```
use Mytest;
```

#### 6.显示本库中的所有表
```
show tables;
```

#### 7.建表
```
#:creat table mytest_acount (col1 INT, col2 CHAR(5), col3 DATE);  表至少一列。
create table 表名 (字段设定列表);
```

#### 8.显示某表的结构
```
describe table1;
```

#### 9.删表
```
drop table 表名;
```

#### 10.修改表名
```
rename table oldname to newname;
```

#### 11.清空表中数据
```
truncate 表名;
```
