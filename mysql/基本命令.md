
## 基本命令
#### 1.显示所有数据库列表
```
show databases;
```
#### 2.建库
```
create database Mytest;
```
#### 3.删库
```
drop database 库名;
```
#### 4.打开某个数据库（比如数据库：Mytest)
```
use Mytest;
```

#### 5.显示本库中的所有表
```
show tables;
```

#### 6.建表
```
#:creat table mytest_acount (col1 INT, col2 CHAR(5), col3 DATE);  表至少一列。
create table 表名 (字段设定列表);
```

#### 7.显示某表的结构
```
describe table1;
```

#### 8.删表
```
drop table 表名;
```
